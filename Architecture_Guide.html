SPC Tool Architecture

A deep dive into the code, flow, and libraries powering the Statistical Process Calculator.

1. System Overview

The SPC Tool is a professional-grade web application built using the Streamlit framework. It serves as a unified platform for statistical process control, combining data analysis, visualization, and automated reporting into a single, high-performance interface.

It operates as a Stateful Web Application where the frontend and backend are tightly coupled via a reactive data flow engine.

2. Technology Stack

Frontend Layer

Rendered in the browser using React (via Streamlit's engine) and Vanilla CSS for custom styling.

Streamlit UI Plotly.js React.js HTML5/CSS3

Backend Layer

A Python 3.12 server handling all statistical computations, file generation, and state management.

Python 3.12 Pandas Numpy Scipy

Reporting & Export

High-quality PDF and Excel generation engines for professional industrial reports.

ReportLab OpenPyXL Kaleido

4. Core Libraries & Usage

Library Primary Purpose
Streamlit Handles the layout, widgets, and the WebSocket connection between Python and the browser.
Pandas The "Data Engine" used for reading Excel/CSV files and managing large datasets in memory.
Scipy.stats Used for complex statistical distributions (Normal, Lognormal) and P-Value calculations.
Plotly Powers the interactive charts, allowing users to hover over points and zoom into data.
ReportLab A low-level PDF generation library that builds the branded Capability Reports.

5. Key Implementation Snippets

The calculation engine is encapsulated in the StatisticalCalculator class to ensure thread safety and modularity.


# Example of Nelson Rules Implementation
def evaluate_nelson_rules(self, data, mean, std):
    rules = {1: [], 2: [], 3: [], 4: [], 5: [], 6: [], 7: [], 8: []}
    for i, v in enumerate(data):
        # Rule 1: Point > 3sigma from mean
        if abs(v - mean) > 3 * std: 
            rules[1].append(i)
        # Rule 2: 9 points same side of mean
        if i >= 8:
            slice_data = data[i-8:i+1]
            if all(x > mean for x in slice_data) or all(x < mean for x in slice_data): 
                rules[2].append(i)
    return rules
