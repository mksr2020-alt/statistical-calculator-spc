SPC Tool Architecture

A deep dive into the code, flow, and libraries powering the Statistical Process Calculator.

1. System Overview

The SPC Tool is a professional-grade web application built using the Streamlit framework. It serves as a unified platform for statistical process control, combining data analysis, visualization, and automated reporting into a single, high-performance interface.

It operates as a Stateful Web Application where the frontend and backend are tightly coupled via a reactive data flow engine.

2. Technology Stack

Frontend Layer

Rendered in the browser or Desktop window using React (via Streamlit's engine) and Vanilla CSS.

Streamlit UI Plotly.js PyWebView HTML5/CSS3

Backend Layer

A Python 3.12 server handling all statistical computations, file generation, and state management.

Python 3.12 Pandas Numpy Scipy

Packaging & Distribution

Advanced bundling techniques to convert Python scripts into standalone Windows executables.

PyInstaller GitHub Actions PowerShell/Bash

3. Deployment Modes

The SPC Tool is designed to be highly versatile, supporting two primary deployment methods:

🌐 Web Mode (Server)

Run on a central PC or server using start_server.bat. Colleagues connect via their browsers using the server's IP address (e.g., http://192.168.1.10:8501).

🖥️ Desktop Mode (.EXE)

A standalone Windows Executable bundled with all dependencies. It runs as a native desktop application with its own window, icon, and splash screen.

6. Desktop Launcher Implementation

To run Streamlit inside a bundled .exe, the launcher manually handles the server boot process and signal patching to allow background execution.


# launcher.py - Booting Streamlit in a daemon thread
def run_streamlit(port: int):
    # Patch signal: handlers can only be set from main thread
    _patch_signal_for_thread()
    
    import streamlit.web.bootstrap as bootstrap
    app_path = resource_path('streamlit_spc.py')
    
    # Configure and start in headless mode
    bootstrap.load_config_options(flag_options={'server.port': port, 'server.headless': True})
    bootstrap.run(app_path, 'streamlit run', [], flag_options)
